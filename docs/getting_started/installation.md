# Installation & Dependencies

Installation is as simple as:

=== "pip"

    ```bash
    pip install odys
    ```

=== "uv"

    ```bash
    uv add odys
    ```

## Install from repository

And if you prefer to install odys directly from the repository:

=== "pip"

    ```bash
    pip install 'git+https://github.com/ramirocrc/odys@main'
    ```

=== "uv"

    ```bash
    uv add 'git+https://github.com/ramirocrc/odys@main'
    ```
